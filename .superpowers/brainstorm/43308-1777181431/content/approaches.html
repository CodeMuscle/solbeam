SolBeam — 3 Architecture Approaches

How we build the data pipeline that powers the signal engine

A

DexScreener-First (Poller)

Use DexScreener as the single source of truth. Poll every 10–30s for new pairs, trending tokens, OHLCV, and volume spikes. Jupiter for live prices. No Helius webhooks needed.

Pros

  • Fastest to build (days not weeks)
  • Zero infrastructure complexity
  • 100% free, no webhook setup
  • DexScreener covers Pump.fun + Raydium + all DEXes

Cons

  • 10–30s discovery lag — miss the very first minute
  • No smart money wallet tracking
  • No on-chain deployer history analysis
  • Weakest signal set of the three
B

Helius-First (Webhook)

Helius webhooks fire the instant a tracked smart money wallet transacts. Near real-time token discovery (<1s lag). DexScreener fills in token metrics after discovery.

Pros

  • Real-time smart money signal — the highest-alpha input
  • Sub-second discovery when a whale enters
  • Deployer history + tx parsing via Helius DAS API
  • Still free (1M credits/mo on Helius free tier)

Cons

  • Only discovers tokens that smart wallets touch — misses organic launches
  • Requires curating a smart wallet list manually
  • Webhook needs live URL (Vercel works fine)
C

⭐ Hybrid Pipeline (Recommended)

Three parallel discovery channels feeding one scoring engine. DexScreener polls for all new/trending tokens. Helius webhooks fire when smart money enters. Jupiter streams live prices for position monitoring.

Pros

  • Covers all three token categories you wanted (new launches + migrations + trending)
  • Smart money signal AND broad token discovery
  • Position monitoring loop for break-even stop-loss alerts
  • All free — Helius + DexScreener + Jupiter + Supabase Realtime
  • Best foundation to later add semi-auto / full-bot mode

Cons

  • More moving parts than A or B
  • Takes 1–2 extra days to wire the pipeline vs Approach A
Why this wins: Smart money entry is the single highest-signal factor for memecoin pumps. Without it, you're just reading candles like everyone else. The hybrid gives you that edge on every token.
